P5.js

Purpose

For this project, I created a replica of my laptop desktop with some of my frequently used apps. Each app icon has its own interaction that is personalized towards me. Through this, I wanted to portray the idea that a person's laptop or phone can tell a lot about who they are and what they're interested in.

Process

To create the individual interactions, I made each a class for each app icon with constructor, display, and clicked functions. In the setup function, I created a new object for each class to then draw to the canvas in the draw function using the display function for each class. Giving each app icon its own class made it easier to differentiate between interactions in the mousePressed function. Since I already had the code for each interaction in the clicked function for each class, I was able to just call the clicked function for each object to get the desired effect. In the demo video, I go further in depth on the interactions for each app icon.
